This project has been divided into two apps namely : MyChatApp and Chrome Extension where:

#1 - MyChatApp is the main web app which is developed using Meteor. It is a simple web app where any user can post there text message which will be instantly displayed to other users accessing the app. To run this app:

a - Install meteor. refer to : http://docs.meteor.com/
b - Execute command : meteor under the MyChatApp repository. This will get the app running on port number 3000.

#2 - ChromeExtension is a simple chrome plugin which refers to MyChatApp via IFrame.
